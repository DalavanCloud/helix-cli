Petri Dish
==========

his is an early POC to see how local development could work w/o running a runtime container.

Setup
-----------

```
# npm link the various modules first
$ npm install

$ cd examples/soupdemo
$ ./setup.sh
$ npm install
$ npm run build
$ npm start

```
